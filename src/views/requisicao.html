JQUERY


URL: viaddd.com.br/api/ddd013 
          
            $.ajax({
              type: "GET",
              url: "http://viaddd.com.br/api/ddd/" + "013",

                  console.log(response)
              }
            });


Formato Fetch

Veja exemplo de uma requisição com Fetch:

JavaScript


URL: viaddd.com.br/api/ddd/013 
          
          fetch("http://viaddd.com.br/api/ddd/" + "013")
            .then((res) => { return res.json(); })
            .then((data) => { console.log(data) })
            .catch((err) => { console.log(err) })


JavaScript (async await + try/catch)


URL: viaddd.com.br/api/ddd/013 
        
          async function getDDD() {
            try{
              const response = await fetch(`http://viaddd.com.br/api/ddd/${013}`)
              const data = await response.json()
              console.log(data)
            }catch(err){
              console.log(err)
            }
          }
